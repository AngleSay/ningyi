# 山西凝一科技有限公司官网 - 设计规划

## 设计理念
**现代工业科技风格** - 融合建筑工业美学与现代科技感

### 核心设计原则
1. **科技感与专业性** - 通过几何形状、现代排版、精准的间距传达专业形象
2. **工业美学** - 运用建筑、工程元素的视觉隐喻
3. **高对比度** - 橙色作为主色调，与白色/深灰色形成强烈对比
4. **功能至上** - 清晰的信息层级，易于导航

## 色彩系统
- **主色** - 橙色 (#FF9500)：代表能量、创新、专业
- **辅助色** - 深灰色 (#1F2937)：代表稳定、专业、信任
- **背景色** - 白色 (#FFFFFF) 和浅灰色 (#F9FAFB)
- **强调色** - 深棕色 (#92400E)：用于重点内容

## 排版系统
- **标题字体** - 思源黑体 Bold (Google Fonts)：现代、有力
- **正文字体** - 思源黑体 Regular：易读、专业
- **字号层级**：
  - 超大标题：48px
  - 大标题：36px
  - 中标题：24px
  - 小标题：18px
  - 正文：16px
  - 辅助文字：14px

## 网站结构
1. **导航栏** - 固定顶部，包含logo、菜单、CTA按钮
2. **Hero Section** - 大背景图 + 品牌标语 + CTA
3. **公司简介** - 三列卡片展示企业定位
4. **核心产品** - 四个产品卡片展示
5. **技术优势** - 数据/特性展示
6. **联系方式** - 页脚信息

## 视觉元素
- 使用几何形状（圆形、方形）作为装饰元素
- 微妙的渐变背景
- 卡片式设计，带阴影和悬停效果
- 响应式布局，移动端友好

## 动画效果
- 滚动时的淡入效果
- 按钮悬停时的缩放和颜色变化
- 卡片的微妙浮动效果
- 平滑的页面过渡

## 响应式设计
- 移动端 (320px - 640px)
- 平板端 (641px - 1024px)
- 桌面端 (1025px+)
